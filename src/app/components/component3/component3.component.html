SwitchMap

Con switchMap podemos cortar un observable. Al hacer click sobre este recuadro narangalo que ocurre en la console es que se inicia un conteo mediante 'interval(1000)'. La gracia de swicthMap es que, al volver a hacer click sobre este recuadro, se reinicia, se rompe, el conteo osea el observable.

A continuación se muestra un contador que se puede detener con el botón Pause y reanudar con el Resume. El funcionamiento es el siguiente: creamos un observable interval y lo mapeamos para que devuelva +1 cada segundo. Para cada botón cramos un obsevable con fromEvent al hacer click. Uno devuelve true y el otro false. Con un "merge" juntamos los dos anteriores observables. Al juntarlos, creamos el observable timer y, con un pipe, hacemos lo siguiente: ponemos como valor inicial true con un "startWicth", luego mediante un switchMap hacemos un if else. Como empezamos con el valor true, devolvemos el valor del observable con interval (suma 1 al contador). Con un scan le sumamos al valor que teníamos antes el +1 del intervalobs. Finalmente, con un takewhile nos aseguramos de que la cuenta no llegue a más de 10. Si le damos al botón Pause, el valor es false y dentro del switchMap devuelve un empty, por lo tanto, el contador se queda como está y se detiene.
